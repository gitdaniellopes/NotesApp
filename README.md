## :bulb: Motivação

A motivação desse projeto foi colocar em pratica o novo kit de UI moderno do android, Jetpack Compose.

## Tecnologias
* 100% Kotlin
* View Model
* Coroutines
* Flow
* Room
* State
* Clean architecture - useCases
